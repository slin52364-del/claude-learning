# 从零星概念到完整框架

"只知道零星几个概念"——这是一个很清晰的起点描述。

这个阶段有一个特有的感受：知道一些词，但不知道这些词之间是什么关系，也不知道自己不知道什么。就像手里有几块拼图，但不知道整张图长什么样。

---

## 一、这个阶段最容易走错的路

**错误方式：继续积累零星概念。**

遇到一个词查一个词，读到一个案例记一个案例——这样做一年，你还是会感觉"知道一些但不成体系"。因为你在往一个没有结构的容器里装东西。

**正确方式：先建骨架，再填内容。**

骨架就是：这个领域的主要问题是什么，分哪几个层次，工作流程大概是什么。

有了骨架，你学到的每个概念都会自动找到位置挂上去，整个知识结构才能立起来。

---

## 二、配网规划的骨架大概长这样

这是一个帮你入门的框架，不是教科书，只是帮你先有个全局：

**第一层：物理结构（电是怎么到用户手里的）**

```
发电厂 → 超高压输电 → 变电站（110kV/35kV）→ 中压配网（10kV线路）→ 配电变压器 → 低压线路（380V/220V）→ 用户
```

配网，指的就是10kV及以下这一段——从变电站出来之后，到用户门口之前，这一段电网的规划和建设。

**第二层：规划在做什么事**

简单说，规划回答的是三个问题：
1. 未来用电需求会长到多少？（负荷预测）
2. 现有电网能不能满足？哪里不够用/不可靠？（现状评估）
3. 需要新建或改造哪些线路和设备？怎么做最合理？（方案设计）

**第三层：一个项目怎么走完**

```
需求识别 → 立项（项目储备）→ 可研（可行性研究）→ 初设（初步设计）→ 施工图 → 施工 → 竣工验收
```

你在规划岗位，主要接触的是前几个阶段：识别需求、论证方案、推动立项。

**第四层：一年的工作节奏**

大概是这样的循环：
- 上半年：执行当年计划、同时开展下一年项目的可研
- 下半年：编制下一年度项目计划、项目储备申报
- 年末：汇总全年成果、准备次年计划

---

## 三、为什么先给你骨架

因为有了这个骨架，你零星知道的那些概念就有地方放了。

比如你知道"环网"这个词，它就挂在"物理结构/接线方式"这一格。你知道"可研"，它在"工作流程"的第三步。原本飘在空中的词，现在有了坐标。

---

## 思考问题

1. 看完这个骨架，你之前知道的那几个概念，现在知道它们大概在哪个位置了吗？（说出来，我们可以一起定位）

2. 你目前工作里，接触最多的是哪个层次的内容？日常碰到的任务，是更偏"方案设计"还是"流程推进"还是别的？

3. 你现在工作里遇到的最让你"听不懂/不知道怎么做"的具体场景是什么？
