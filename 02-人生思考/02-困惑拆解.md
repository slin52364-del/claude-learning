# 困惑拆解

> 很多困惑之所以长期悬而未决，是因为我们从来没有认真把它说清楚。
> 这份模板帮你把模糊的困惑变成可以处理的问题。

---

## 一、把困惑写出来

> 不加修饰，用最原始的语言把它写下来。哪怕语无伦次也没关系。

```
我最近一直想不清楚的是……




```

---

## 二、这个困惑的本质是什么？

> 困惑通常是多种问题混在一起的。把它拆开，找到真正的核心。

勾选你的困惑属于哪类（可多选）：

- [ ] **方向问题** — 我不知道该去哪，或者该做什么
- [ ] **价值判断问题** — 我不知道什么更重要，或哪个选择更好
- [ ] **信息不足问题** — 我还没有足够的信息来决定
- [ ] **情绪问题** — 我在情绪中，还没办法理性思考
- [ ] **执行问题** — 我知道该做什么，但做不到
- [ ] **关系问题** — 和某个人之间的事让我纠结
- [ ] **认知问题** — 我的某个信念或假设可能有问题

**核心是：**

```
（一句话提炼困惑的本质……）
```
