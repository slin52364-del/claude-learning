# 07. 故障指示器——怎么知道故障在哪一段

## 问题还原

保护动作之后，线路跳闸了。

调度知道的信息：某条线路的出口断路器跳了。

不知道的：故障在这条线路的哪一段？

没有辅助手段，只能派人沿线巡视。10 公里的线路，几十个档距，人走下来至少半天。

故障指示器解决的就是这个问题：**不用全线找，直接告诉你故障在哪里。**

---

## 核心原理：故障电流留下的"痕迹"

短路故障发生时，电流从变电站出发，沿着线路流向故障点，然后通过故障点入地（或相间短路）。

关键在于：**故障点之后的线路，没有故障电流流过。**

```
变电站
  │
  ├── 故障电流流过 ── 故障电流流过 ──★故障点★
  │                                       │
  │                              故障点之后：无故障电流
  └──（末端方向）
```

故障指示器就是检测"这里有没有故障电流流过"的设备。

把它安装在线路各段：

- 故障点之前的指示器：**触发**（检测到故障电流）
- 故障点之后的指示器：**未触发**（没有故障电流）

最后一个触发的指示器和第一个未触发的指示器之间，就是故障区段。

---

## 故障指示器怎么判断"故障电流"

正常运行时线路也有电流，指示器怎么区分正常电流和故障电流？

靠两个特征：

**特征一：幅值突变**

故障电流比正常负荷电流大很多，几倍到几十倍。指示器内部设一个门槛，超过这个值才认为是故障电流。

**特征二：线路失压**

短路故障之后，保护动作，线路跳闸，线路电压消失。指示器检测到"先有大电流冲击，紧接着线路失压"，确认这是一次故障，翻转标志。

这两个条件组合判断，能有效避免误动（比如电机启动时的大电流冲击，之后线路并没有失压）。

---

## 两种类型

### 翻转型（传统型）

指示器上有一个机械指示牌，正常时是某个颜色（如白色），故障触发后翻转成另一个颜色（如红色）。

人到现场看，红的就是触发了的。

**优点**：便宜、可靠、不需要电源
**缺点**：必须人到现场才能看到，没有节省巡线的时间，只是到了现场后定位更快一点

复归方式：线路重新送电后，指示器会自动复归（或人工按一下）。

### 通信型（主流）

指示器触发后，通过无线通信（4G/LoRa/GPRS）把触发信息发送给配电主站。

调度在屏幕上就能看到哪个指示器动了，不用人跑现场。

**优点**：远程可视，故障定位快，可以和配电自动化系统联动
**缺点**：需要通信覆盖，有一定成本，需要维护电池或取能装置

---

## 安装位置怎么选

不是越多越好，多了成本高、维护量大，还可能带来误报干扰。

一般原则：

**装在分段开关和分支开关旁边**

这样每个可操作的区段都有对应的指示器，定位结果和操作单元直接对应。

```
变电站
  │
  [FI] ── [分段开关A] ── [FI] ── [分段开关B] ── [FI] ── 末端
                │                      │
             [FI]                   [FI]
           [分支开关1]            [分支开关2]
              │                      │
           分支1                   分支2
```

故障发生后，哪两个相邻的 FI 之间"一个触发一个没触发"，故障就在那里。

**分支上也要装**

主干线 FI 能确定故障在哪个分支入口和哪个分段之间，但分支内的故障还需要分支上的 FI 进一步定位。

---

## 实际场景走一遍

线路结构：

```
变电站 ──[FI1]──[分段A]──[FI2]──[分段B]──[FI3]──末端
                              │
                           [FI4]
                         [分支开关]
                              │
                            分支（配变若干）
```

分支发生故障：

1. 故障电流从变电站流出，经过 FI1、FI2、FI4，到达故障点
2. FI3 没有故障电流（故障在分支上，电流没有走到 FI3 那里）
3. 保护动作，跳闸
4. FI1、FI2、FI4 触发；FI3 未触发
5. 调度判断：FI4 触发说明故障在分支内；分支开关下游故障
6. 操作：断开分支开关，隔离分支；重合变电站断路器，主干恢复送电
7. 分支内故障段等人工处理

**整个定位过程：秒级（通信型）或"到现场就知道"（翻转型）**

---

## 故障指示器的局限

- **只能定段，不能定点**：告诉你故障在哪两个指示器之间，但不能告诉你具体是哪根电线、哪个位置
- **依赖通信**：通信型指示器如果通信中断，跟没有一样
- **误报问题**：整定不合理或设备老化，会出现误触发，反而干扰判断
- **对复杂故障的局限**：高阻接地故障（如树枝轻搭导线）故障电流很小，可能检测不到

---

## 和保护的分工

| | 保护 | 故障指示器 |
|--|------|-----------|
| 动作时间 | 毫秒级 | 跳闸后上报，秒级 |
| 作用 | 切除故障，保护设备 | 定位故障，指导操作 |
| 动作结果 | 开关跳闸 | 触发翻转/上报信息 |
| 没有它会怎样 | 故障扩大，设备损坏 | 靠人工巡线定位，慢 |

两者不替代，配合使用：保护负责"快速出局"，指示器负责"找到尸体在哪里"。
