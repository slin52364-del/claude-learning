# 06. 保护怎么动作的——从电气量到跳闸指令

## 你的理解对不对

你说的：

> "通过判断开关的电气量异常程度来控制开关动作，管开关后面的线路状况，让故障能在整体范围内快速出局"

这个理解**基本准确**，补充两点：

1. 监测的不是"开关本身的电气量"，而是开关**安装位置处**的线路电气量（通过互感器采集）
2. "整体范围内快速出局"这句话值得展开——不同位置的保护，动作速度故意设计成不一样快，这是保护能"精确出局"而不是"乱出局"的关键

---

## 保护判断的是什么电气量

10kV 配电线路的保护，主要盯两个量：

**电流（最主要）**

正常运行时，线路电流在一个合理范围内（几十安到几百安，视负荷大小）。

短路发生时，电流暴增，可能达到正常值的十几、几十倍。这个变化非常剧烈，是最可靠的故障判据。

**电压（辅助）**

短路时，电压会大幅跌落。电压判据常用来辅助确认故障，或用于特定场景的保护逻辑，但不是配电线路保护的主角。

---

## 保护怎么"判断"——整定值的概念

保护装置里有一个预先设好的门槛值，叫**整定值**。

逻辑很简单：

```
采集到线路电流
  │
  ▼
当前电流 > 整定值？
  ├── 否 → 什么都不做，继续监测
  └── 是 → 启动计时器
              │
              ▼
         计时到了？
              ├── 否 → 电流已经回去了（瞬间扰动），复归，不动作
              └── 是 → 发跳闸命令给断路器
```

两个整定参数决定了保护的行为：

- **电流整定值**：超过多大的电流才认为是故障
- **时间整定值**：超过多久才跳闸（时限）

---

## 两种主要保护类型

### 速断保护（瞬时电流速断）

**整定逻辑**：电流超过一个**很高的门槛**（通常是最大负荷电流的好几倍），**立刻跳闸**，不等待。

**为什么要这么高的门槛？**

因为这个保护是"无选择性"的——它一动作，整个开关就跳了。为了避免在正常的启动电流、冲击电流时误动，门槛定得很高。

**能保护的范围**：只能保护线路靠近电源侧的一段（因为越远处短路，电流越小，可能达不到这个高门槛）。

**优点**：快，不延时，对电气设备的冲击最小。

---

### 过流保护（定时限过流）

**整定逻辑**：电流超过一个**较低的门槛**（比负荷电流大一些但不像速断那么高），**等一段时间再跳闸**。

**时间为什么不是零？**

这里涉及配电网保护最关键的概念——**级差配合**（也叫选择性配合）。

---

## 级差配合：为什么保护要"有快有慢"

假设一条线路有这样的结构：

```
变电站出口断路器
  │
  ├─[分支开关1]── 分支1
  │
  ├─[分支开关2]── 分支2 ──── ★故障★
  │
  └─[主干末端]
```

分支2 发生故障，我们**希望**的结果是：只有分支开关2跳闸，分支1和主干继续供电。

**怎么实现这个"只有2跳"？**

靠时间差：

| 开关 | 时间整定值 |
|------|-----------|
| 分支开关2（靠近故障） | 0.5 秒 |
| 分支开关1 | 1.0 秒 |
| 变电站出口断路器 | 1.5 秒 |

故障发生后：
- 0.5秒：分支开关2跳闸，切除故障
- 故障切除了，分支开关1和变电站出口的计时器复归，不动作
- 结果：只有分支2停电 ✓

**如果分支开关2因为某种原因没跳**：
- 1.5秒：变电站出口断路器跳，全线停电（但这是最后的兜底）

这就是级差配合的本质：**离故障越近的开关，动作越快；离故障越远（越靠近电源）的开关，动作越慢，作为后备**。

每一级之间留出的时间差，就叫**级差**，一般取 0.3～0.5 秒（确保下一级开关已经动作完毕）。

---

## 重合闸：跳了之后还要试一次

线路跳闸后，保护装置（或断路器控制回路）会自动发一个合闸命令，**重新合上断路器**。

等一下，刚跳了为什么要合？

因为配电线路上很多故障是**瞬时性的**：

- 树枝碰线一下，放电，就没了
- 鸟粪落在绝缘子上，击穿放电，冲走了就好了
- 雷击感应过电压，过了就过了

这类故障，跳闸之后电弧熄灭，绝缘自然恢复，重合一次就能继续供电。

**重合闸的流程：**

```
线路故障 → 保护跳闸 → 等待 0.5～1 秒（让电弧熄灭）→ 重合
  │
  ├── 重合成功（瞬时故障）→ 正常运行，用户最多感觉到一次短暂闪灯
  └── 重合后再跳（永久故障）→ 闭锁，不再重合，等人工处理
```

实际运行中，重合成功率很高（70%以上），所以重合闸是非常有价值的功能。

---

## 把动作机制串起来

```
故障发生
  │
  ▼
互感器采集到故障电流，传给保护装置
  │
  ▼
保护装置判断：超过整定值？
  │
  ▼
启动计时（按级差配合，各级开关各自计时）
  │
  ▼
最近的开关先到时间 → 发跳闸令 → 断路器断开 → 故障切除
  │
  ▼
（如有重合闸）延时后发合闸令
  │
  ├── 成功 → 恢复运行
  └── 再次过流 → 闭锁，不再重合
```

---

## 你的理解补完了吗

你说的"判断电气量异常程度来控制开关动作"——**准确**，但更完整的说法是：

> 保护装置采集开关安装处的电气量（主要是电流），与整定值比较；超过整定值并持续一段时间后，发跳闸命令给断路器。多级保护通过时间级差配合，确保离故障最近的开关先跳，实现有选择性的故障切除。

你说的"管开关后面的线路状况"——**也准确**，这正是"保护配在开关上，保护的是开关所控制的线路区段"这句话的含义。

## 可能还不清楚的地方

- **互感器是什么？** → 专门负责采集电流/电压信号的设备，因为线路上的实际电流太大，无法直接送进保护装置，互感器把它等比例缩小成可以处理的信号
- **整定值怎么算出来的？** → 有专门的继电保护整定计算，是一套独立的工程计算工作，规划阶段不需要深入，知道"有这个计算过程"就够
- **配电自动化（FA）和保护有什么关系？** → FA 是在保护动作之后的"第二层"：保护负责切除故障，FA 负责定位、隔离、恢复。这是下一个值得展开的方向
