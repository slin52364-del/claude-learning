# 06. 保护与故障指示器的工作机制——从电气量到故障定位

## 这篇讲什么

前几篇建立了故障处理的系统框架：**跳闸 → 定位 → 隔离 → 转供**。

这篇往深里走一层，把前两步的机制讲透：

- **保护怎么判断有故障、怎么跳闸的**
- **故障指示器怎么知道故障电流经过这里**

两者是配合关系：保护负责"快速出局"，故障指示器负责"找到尸体在哪里"。

---

## 第一部分：保护的工作机制

### 保护盯的是什么电气量

10kV 配电线路的保护，主要盯两个量：

**电流（最主要）**

正常运行时，线路电流在合理范围内（几十安到几百安）。短路发生时，电流暴增，可能达到正常值的十几、几十倍。这个变化非常剧烈，是最可靠的故障判据。

**电压（辅助）**

短路时电压会大幅跌落，常用来辅助确认故障，但不是主角。

> **互感器的作用**：线路实际电流太大，无法直接送进保护装置。互感器把它等比例缩小成可处理的信号，再送给保护装置判断。

---

### 整定值：保护的判断门槛

保护装置里预先设好一个门槛值，叫**整定值**。判断逻辑：

```
采集到线路电流
  │
  ▼
当前电流 > 整定值？
  ├── 否 → 继续监测
  └── 是 → 启动计时器
              │
              ▼
         持续超过设定时间？
              ├── 否 → 电流已回落（瞬间扰动），复归，不动作
              └── 是 → 发跳闸命令给断路器
```

两个整定参数决定保护行为：

- **电流整定值**：超过多大的电流才认为是故障
- **时间整定值**：超过多久才跳闸

---

### 两种主要保护类型

**速断保护（瞬时电流速断）**

电流超过一个**很高的门槛**，立刻跳闸，不等待。

门槛设得高，是为了避免在正常启动电流、冲击电流时误动。代价是只能保护线路靠近电源的一段（越远处短路电流越小，可能够不到门槛）。

**优点**：快，对设备冲击最小。

---

**过流保护（定时限过流）**

电流超过一个**较低的门槛**，等一段时间再跳闸。

为什么不立刻跳？这里涉及配电网保护的关键机制——**级差配合**。

---

### 级差配合：为什么保护要"有快有慢"

假设分支2发生故障，希望的结果是：**只有分支开关2跳，其他继续供电**。

```
变电站出口断路器
  │
  ├─[分支开关1]── 分支1
  ├─[分支开关2]── 分支2 ──── ★故障★
  └─[主干末端]
```

靠时间差实现：

| 开关 | 时间整定值 |
|------|-----------|
| 分支开关2（靠近故障） | 0.5 秒 |
| 分支开关1 | 1.0 秒 |
| 变电站出口断路器 | 1.5 秒 |

故障发生后：
- 0.5秒：分支开关2跳闸，故障切除
- 分支开关1和变电站出口的计时器复归，不动作
- 结果：只有分支2停电 ✓

**如果分支开关2没跳**：变电站出口在1.5秒后跳，全线停电（兜底保护）

**本质**：离故障越近的开关动作越快，越靠电源侧的越慢，作为后备。每一级之间留出的时间差叫**级差**，一般取 0.3～0.5 秒。

---

### 重合闸：跳了之后还要试一次

线路跳闸后，保护会自动发一个合闸命令，重新合上断路器。

为什么？因为配电线路上很多故障是**瞬时性的**：

- 树枝碰线一下，放电就没了
- 鸟粪落在绝缘子上，冲走就好了
- 雷击感应过电压，过了就过了

这类故障，跳闸后电弧熄灭，绝缘自然恢复，重合一次就能继续供电。

```
线路故障 → 保护跳闸 → 等 0.5～1 秒（让电弧熄灭）→ 重合
  │
  ├── 重合成功（瞬时故障）→ 正常运行，用户感觉到一次短暂闪灯
  └── 重合后再跳（永久故障）→ 闭锁，不再重合，等人工处理
```

实际运行中重合成功率很高（70%以上），是非常有价值的功能。

## 第二部分：故障指示器的工作机制

### 核心原理：故障电流留下的"痕迹"

短路故障发生时，电流从变电站流向故障点，然后通过故障点入地。

**关键**：故障点之后的线路，没有故障电流流过。

```
变电站
  │
  ├── 故障电流流过 ── 故障电流流过 ──★故障点★
  │
  │                              故障点之后：无故障电流
  └──（末端方向）
```

故障指示器安装在线路各段：

- 故障点之前的指示器：**触发**（检测到故障电流）
- 故障点之后的指示器：**未触发**（没有故障电流）

最后一个触发的和第一个未触发的之间，就是故障区段。

---

### 怎么区分正常电流和故障电流

靠两个特征组合判断：

**特征一：幅值突变**

故障电流比正常负荷电流大很多（几倍到几十倍），内部门槛超过才认为是故障电流。

**特征二：线路失压**

短路后保护动作、线路跳闸，电压消失。"先有大电流冲击，紧接着失压"——确认是一次故障，翻转标志。

两个条件组合，能有效避免误动（比如电机启动的大电流，之后线路并没有失压）。

---

### 两种类型

**翻转型（传统型）**

指示器上有机械指示牌，触发后翻转颜色（如白→红）。人到现场看颜色。

**优点**：便宜、可靠、不需要电源
**缺点**：必须人到现场才能看到，节省的是"知道去哪里"的时间，不省路程

---

**通信型（主流）**

触发后通过无线通信（4G/LoRa/GPRS）把触发信息发给配电主站，调度在屏幕上直接看哪个指示器动了。

**优点**：远程可视，故障定位快，可和配电自动化联动
**缺点**：需要通信覆盖，有成本，需维护电池或取能装置

---

### 安装位置怎么选

**装在分段开关和分支开关旁边**——每个可操作区段都有对应指示器，定位结果和操作单元直接对应。

```
变电站
  │
  [FI] ── [分段开关A] ── [FI] ── [分段开关B] ── [FI] ── 末端
                │                      │
             [FI]                   [FI]
           [分支开关1]            [分支开关2]
              │                      │
           分支1                   分支2
```

哪两个相邻 FI 之间"一个触发一个没触发"，故障就在那里。**分支上也要装**，才能定位分支内部的故障。

---

### 实际场景走一遍

分支发生故障：

```
1. 故障电流从变电站流出，经过 FI1、FI2、FI4，到达故障点
2. FI3 未触发（电流没走到那里）
3. 保护动作，跳闸
4. 调度判断：FI4 触发 → 故障在分支内
5. 断开分支开关，隔离分支
6. 重合变电站断路器，主干恢复送电
7. 分支内故障段等人工处理
```

---

### 故障指示器的局限

- **只能定段，不能定点**：告诉你故障在哪两个指示器之间，不能精确到具体位置
- **依赖通信**：通信型如果通信中断，跟没有一样
- **误报问题**：整定不合理或设备老化，会误触发，干扰判断
- **高阻接地盲区**：树枝轻搭导线等高阻故障，电流很小，可能检测不到

---

## 两者的分工

| | 保护 | 故障指示器 |
|--|------|-----------|
| 动作时间 | 毫秒级 | 跳闸后上报，秒级 |
| 作用 | 切除故障，保护设备 | 定位故障，指导操作 |
| 动作结果 | 开关跳闸 | 触发翻转 / 上报信息 |
| 没有它会怎样 | 故障扩大，设备损坏 | 靠人工巡线定位，慢 |

两者不替代，配合使用：保护负责"快速出局"，指示器负责"找到尸体在哪里"。

3. 故障指示器的密度怎么定合适——装多了有成本，装少了定位不准，决定这个密度的因素是什么？
