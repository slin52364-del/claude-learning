# 02. 10kV 配电网典型接线方式

## 为什么从这里开始

你的工作是 10kV 线路网架优化。网架优化的本质，就是在现有条件下，把接线方式调整到更合理的状态。

所以，**先把几种典型接线方式的结构、优缺点、适用场景搞透彻**，是一切后续工作的前提。

---

## 三种基本接线方式

### 1. 放射式（辐射式）

```
变电站
  │
  ├── 线路 A ──── 配变1 ── 配变2 ── 配变3
  │
  └── 线路 B ──── 配变4 ── 配变5
```

**结构特点：** 从变电站出线，单条线路串联若干配变，末端开路，无联络。

**优点：**
- 结构简单，建设成本低
- 保护整定容易，故障定位直接

**缺点：**
- 可靠性最差：线路任何一点故障，该线路后端全停
- 无法转供，故障恢复时间长

**适用场景：** 农村、偏远地区、负荷密度低、对可靠性要求不高的场合。

---

### 2. 环式（手拉手）

```
变电站 A                    变电站 B
    │                           │
    └── 线路 A1 ──┬── 线路 B1 ──┘
                  │
              常开联络开关（NO）
```

**结构特点：** 两条线路（可来自同一变电站不同母线，或不同变电站）末端通过一个**常开开关**相连，构成"开环运行、闭环设计"。

**正常运行：** 联络开关断开，两条线路各自独立供电。

**故障时：** 隔离故障段，合上联络开关，由另一条线路转供非故障段负荷。

**优点：**
- 可靠性大幅提升：单条线路故障可转供
- 满足 N-1 校验要求（这是规划的基本门槛）
- 是城市配电网的主流接线方式

**缺点：**
- 需要联络开关和相应的自动化配合
- 转供时需人工或自动操作，有一段停电时间

**适用场景：** 城市中压配电网主干线，工业园区，重要用户集中区域。

**关键概念 — N-1 原则：**
> 电网中任意一个元件（线路、变压器）故障退出运行后，通过切换操作，其余设备能在不过载的前提下恢复对所有用户供电。
> 手拉手接线的设计目标就是满足这个要求。

---

### 3. 多分段多联络

```
变电站 A                              变电站 B
    │                                     │
    └─ 分段开关 ─┬─ 分段开关 ─┬─ 分段开关 ─┘
                 │             │
            联络开关①      联络开关②（均常开）
                 │             │
    变电站 C ────┘             └──── 变电站 D
```

**结构特点：** 一条线路被分段开关分成若干段，每段末端（或中间）与其他线路有联络，形成多个备用转供路径。

**常见形式：**
- **两分段两联络**：一条线路分2段，有2个联络点
- **三分段三联络**：分3段，3个联络点（国网典型设计推荐）

**优点：**
- 故障时只需停最小范围（某一分段），其余可快速转供
- 供电可靠性更高，与配电自动化结合后可实现故障自愈
- 负荷分配更灵活

**缺点：**
- 接线复杂，设备多，建设成本高
- 对自动化系统依赖度高
- 运行管理难度增加

**适用场景：** 城市核心区、高负荷密度区域、对可靠性要求高的区域（A+类、A类供电区）。

## 三种接线方式对比

| 对比维度 | 放射式 | 手拉手环式 | 多分段多联络 |
|---------|--------|-----------|------------|
| 可靠性 | 低 | 中 | 高 |
| 建设成本 | 低 | 中 | 高 |
| 运维复杂度 | 低 | 中 | 高 |
| N-1 满足 | 否 | 是 | 是 |
| 适用区域 | 农村/偏远 | 城市一般区域 | 城市核心区 |
| 配自需求 | 无 | 建议有 | 必须有 |

## 网架优化的核心逻辑

你做的是**网架优化**，本质上是在问：

> 现有的接线方式，是否匹配该区域的负荷密度、可靠性要求和投资水平？

常见的优化方向：

1. **放射式 → 手拉手**：提升可靠性，满足 N-1
2. **增加分段开关**：缩小故障停电范围
3. **新增联络**：提高转供能力
4. **消除过载**：调整线路负载率到合理区间（一般不超过 70%）
5. **缩短供电半径**：新增电源点，解决末端电压质量问题
