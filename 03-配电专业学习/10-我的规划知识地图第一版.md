# 你的知识地图（第一版）：已知比你以为的多

## 先说"常规工作不算"这件事

你说报销、汇总材料、月度总结"都是很常规的工作，没有取得什么好成绩，不算"。

这个"不算"本身值得拆一下：你在用什么标准判断？

如果标准是"非常规的、有亮眼成绩的才算完成"，那就等于给自己设了一个几乎永远不会触发的成就感开关——因为大多数工作的大多数时候，都是常规的。

常规工作做好，有几件事是真实发生的：流程没有卡住、别人不用替你收尾、下次同样的事你不需要再想怎么做。这些都是价值，只是不够显眼。

真正"没有成绩"的常规工作，和"做好了但不显眼"的常规工作，感觉是不一样的。你自己会知道区别。

### 二、电网结构知识（你已经掌握）

**电压层级**
- 变电站 → 变压器 → 用户
- 10kV（配电网侧）→ 220V（用户侧）

**网架结构**
- 主干线 + 分支线
- 分段：通过开关 / 配电柜 / 分支箱实现
- 联络：故障或检修时，只停一小块，其他区域通过联络线继续供电

**智能化设备**
- 故障指示器：定位故障位置
- 智能开关：支持线上操作、远程控制

---

### 三、评判标准（你已经掌握）

| 指标 | 含义 |
|------|------|
| 线路联络率 | 每条线不能是单辐射，必须与其他线联络 |
| 有效联络率 | 抽掉一个联络节点，仍能正常供电 |
| 站间联络 | 联络的两条线最好来自不同的变电站 |
| 定义标准 | 主干线2-5分段，每分段不超过1个联络 |
